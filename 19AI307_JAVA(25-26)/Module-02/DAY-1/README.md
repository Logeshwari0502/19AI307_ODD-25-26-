# Ex.No:2(A) CLASS AND OBJECT

## QUESTION:
Write a class Animal with a method speak() that prints "I am an animal". Create an object and call the method.

## AIM:
To write a Java class with a method that prints a message and to create an object to call the method.

## ALGORITHM :
1. Start the program and define a class (e.g., prog).
2. Inside the class, define the method speak() that prints "I am an animal".
3. In the main method, create an object of the class using new.
4. Call the speak() method using the created object.
5. End the program.

## SOURCE CODE:
```
class prog{
    void speak(){
        System.out.println("I am an animal");
    }
    
    public static void main(String args[]){
        prog a=new prog();
        a.speak();
    }
}
```
